# PRINT_TEXT_C
This is a program that prints the text you provide as an argument to the STDOUT; the text is colored!

## Usage
For help page:
``` shell
>./PRINT_N
```
Basic usage:
``` shell
>./PRINT_N <Text>
```
Example:
``` shell
>./PRINT_N hakam
```
The above Command will output:
``` shell

   ##    ##     ##     ##    ##     ##     ##    ##
   ##    ##    ####    ##  ##      ####    ###  ###
   ########   ##  ##   ####       ##  ##   ## ## ##
   ##    ##  ########  ##  ##    ########  ##    ##
   ##    ##  ##    ##  ##    ##  ##    ##  ##    ##
```
Another example:
``` shell
>./PRINT_N "hakam 1337"
```
The above Command will output:
``` shell

   ##    ##     ##     ##    ##     ##     ##    ##
   ##    ##    ####    ##  ##      ####    ###  ###
   ########   ##  ##   ####       ##  ##   ## ## ##
   ##    ##  ########  ##  ##    ########  ##    ##
   ##    ##  ##    ##  ##    ##  ##    ##  ##    ##


   ####     ######   ######   #######
     ##          ##       ##      ##
     ##       ####     ####      ##
     ##          ##       ##    ##
   #######  ######   ######    ##
```
Usage with Colors:
``` shell
>./PRINT_N <Text> <color>
```
Example:
``` shell
>./PRINT_N "hakam 1337" red
```
